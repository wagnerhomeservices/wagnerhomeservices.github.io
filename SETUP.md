# Wagner Home Services website setup

1. Upload every file and folder in this package to the root of the GitHub Pages repository.
2. Keep `.nojekyll`, `CNAME`, `index.html`, `updates.html`, `style.css`, and `script.js` at the repository root.
3. To activate the estimate form, deploy `google-apps-script.gs` as a Google Apps Script web app and paste its `/exec` URL into `appsScriptUrl` in `script.js`.
4. Confirm that `CNAME` contains `whomeservices.net`.
5. After deployment, test the phone link, estimate form, mobile navigation, project filters, and review carousel.
